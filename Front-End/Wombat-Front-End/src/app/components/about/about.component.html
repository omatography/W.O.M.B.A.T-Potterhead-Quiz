W.O.M.B.A.T is a web application based on a quiz which proves whether you are a true potterhead or not. Just try out the quiz, you will feel amazing.

The tools and frameworks which has been used here:

  • Frontend: Angular
  • Backend: Spring Boot
  • Database: Mysql
  • ApiTesting: Postman
  • CI and CD: Jenkins
